# Simple Procurement System

Fullstack procurement system built with:
- Go Fiber (Backend)
- MySQL
- HTML + Tailwind + jQuery (Frontend)
- Docker Compose

## Structure
- /backend → API
- /frontend → Web UI

## Run
git clone https://github.com/okanfdlh/Simple-Procurement-System.git
cd Simple-Procurement-System
docker compose up -d --build
